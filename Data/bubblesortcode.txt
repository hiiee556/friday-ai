**Bubble Sort Algorithm**

Bubble sort is a simple sorting algorithm that works by repeatedly iterating through a list of items, comparing adjacent items and swapping them if they are in the wrong order. The algorithm continues to iterate through the list until no more swaps are needed, indicating that the list is sorted.

**Code Implementation**

Here is an example implementation of the bubble sort algorithm in Python:
```python
def bubble_sort(arr):
    """
    Sorts an array in ascending order using the bubble sort algorithm.

    Args:
        arr (list): The array to be sorted.

    Returns:
        list: The sorted array.
    """
    n = len(arr)

    for i in range(n-1):
        # Create a flag that will allow the function to terminate early if there's nothing left to sort
        swapped = False

        for j in range(n-i-1):
            # Compare the adjacent items
            if arr[j] > arr[j+1]:
                # Swap the items if they are in the wrong order
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True

        # If no swaps were made in the inner loop, the array is already sorted
        if not swapped:
            break

    return arr

# Example usage
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
print("Sorted array:", bubble_sort(arr))
```
**Output**
```
Original array: [64, 34, 25, 12, 22, 11, 90]
Sorted array: [11, 12, 22, 25, 34, 64, 90]
```
**Explanation**

1. The outer loop `for i in range(n-1)` iterates `n-1` times, where `n` is the length of the array.
2. The inner loop `for j in range(n-i-1)` compares each pair of adjacent items in the array.
3. If an item is greater than its adjacent item, they are swapped using tuple assignment `arr[j], arr[j+1] = arr[j+1], arr[j]`.
4. The `swapped` flag is used to terminate the algorithm early if no swaps are made in a pass.
5. The algorithm returns the sorted array.

**Time Complexity**

The time complexity of bubble sort is O(n^2) in the worst case, where `n` is the length of the array. However, in the best case (when the array is already sorted), the time complexity is O(n).

**Space Complexity**

The space complexity of bubble sort is O(1), as it only requires a constant amount of additional memory to store the temporary swap values.
